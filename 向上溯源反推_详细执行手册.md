# 向上溯源反推系统：详细执行手册 (Execution Manual)

本文档是将架构设计转化为代码落地的**操着手册**。请按照以下步骤逐一执行。

## 0. 准备工作 (Pre-requisites)

*   **硬件环境**: 建议使用 NVIDIA 显卡 (需运行 VLM)，或准备 API Key (DeepSeek / OpenAI)。
*   **开发环境**: Python 3.10+, PyTorch, FFmpeg (处理视频)。
*   **文件结构**:
    ```text
    /project_root
    ├── /data
    │   ├── /raw_videos       # 原始教学视频
    │   ├── /keyframes        # 抽取的关键帧
    │   └── /golden_dataset   # 标注好的“优质视频”数据
    ├── /src
    │   ├── /perception       # 感知层代码
    │   ├── /agents           # Agent 提示词与逻辑
    │   └── /utils            # 工具脚本
    └── main.py
    ```

---

## 阶段一：数据工程与感知层搭建 (Data & Perception)

**目标**: 让机器“看懂”视频里的教学元素。

### 步骤 1.1：视频切片与抽帧 (Video Processing)
*   **任务**: 编写脚本，将长视频按“镜头”或“场景”自动切分，并提取关键帧。
*   **技术点**: 使用 `PySceneDetect` 库识别场景切换。
*   **代码文件**: `src/utils/video_processor.py`
    *   `def extract_scenes(video_path):` -> 输出场景列表 (start_time, end_time)
    *   `def save_keyframe(scene):` -> 保存 .jpg 图片

### 步骤 1.2：感知层 Schema 定义 (Schema Definition)
*   **任务**: 定义 VLM 需要看什么。这不仅仅是 Caption，而是结构化数据。
*   **输出格式 (JSON)**:
    ```json
    {
      "visual_elements": ["Red arrow on map", "Zoom-in transition"],
      "text_content": ["Title: Photosynthesis"],
      "scaffolding_cues": ["Highlighting key terms", "Schematic diagram"]
    }
    ```

### 步骤 1.3：VLM 接口封装 (VLM Client)
*   **任务**: 调用多模态模型 (如 GPT-4o 或 LLaVA) 分析关键帧。
*   **提示词策略**:
    > "你是一位教育技术专家。请分析这张教学视频截图。列出画面中所有的‘视觉支架’（如箭头、高亮）、教学内容的呈现方式（实拍/动画），以及当前的镜头语言。"
*   **代码文件**: `src/perception/vlm_analyzer.py`

---

## 阶段二：推理层 Agent 开发 (Reasoning Agents)

**目标**: 让机器“理解”设计意图并反推参数。

### 步骤 2.1：教学专家 Agent (Pedagogy Expert)
*   **任务**: 从视觉描述推断教学法。
*   **Prompt 模板**: `src/agents/pedagogy_agent.py`
    *   输入: "画面显示：慢动作回放篮球投篮手部细节，并在手腕处有红色圆圈标注。"
    *   输出: "教学意图：强调关键动作细节 (Key Skill Emphasis)；策略：视觉提示 (Visual Cueing)。"

### 步骤 2.2：艺术总监 Agent (Art Director)
*   **任务**: 提取风格参数。
*   **Prompt 模板**: `src/agents/art_agent.py`
    *   输入: "画面风格：极简线条，淡蓝色背景，高对比度文字。"
    *   输出: "Style Parameters: Minimalist vector art, soft cool lighting, high legibility."

### 步骤 2.3：结构化提示词合成 (Prompt Synthesis)
*   **任务**: 将上述 Agent 的输出拼装成 AIGC Prompt。
*   **代码文件**: `src/agents/prompt_synthesizer.py`
    *   逻辑: `Final_Prompt = [Style] + [Subject] + [Action] + [Camera] + [Lighting]`

---

## 阶段三：黄金模式挖掘 (Pattern Mining)

**目标**: 找出“好提示词”的规律。

### 步骤 3.1：构建对比数据集
*   **任务**: 准备两组视频。
    *   Group A: 获奖/高播放量的优质教学视频。
    *   Group B: 普通/枯燥的教学视频。

### 步骤 3.2：差异化分析
*   **任务**: 对 A 和 B 分别运行上述反推流程，得到 `Prompt_List_A` 和 `Prompt_List_B`。
*   **分析脚本**: `src/analysis/pattern_miner.py`
    *   使用 TF-IDF 或 Embedding 聚类，找出 Group A 独有的词汇（如 `cinematic lighting`, `exploded view`, `4k macro`）。

## 阶段四：应用与验证 (Validation)

### 步骤 4.1：生成测试
*   使用反推出来的“黄金提示词”在 Midjourney/Sora 中生成新视频。

### 步骤 4.2：人工评估
